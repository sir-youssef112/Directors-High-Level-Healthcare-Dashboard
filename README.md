📌 Project Overview

This project presents an executive-level Power BI dashboard designed to monitor healthcare business performance across multiple departments.

The dashboard provides directors with a comprehensive view of Revenue, OPD Volume, CPV, Marketing, and CRM performance by comparing Actual, Baseline, Historical, and Target values.
